# camila-admin

Contiene un administrador de servicios (UI)

## Pre-condiciones

* JDK >= 21
* Docker >= 24.0.6
* maven >= 3.9.4
* Spring >= 6.x
* Spring-boot >= 3.2.x

## Enlaces

* UI (dev): <http://localhost:8100/>
* Actuator:
  * <http://localhost:8100/actuator/health>
